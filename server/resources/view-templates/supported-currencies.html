Make Some Money Stocks Currencies

Supported Currencies

This is a list of currencies used by the machine learning model to predict stock prices.

  • Australian Dollar (AUD)
  • Canadian Dollar (CAD)
  • Chinese Yuan (CNY)
  • Euro (EUR)
  • Japanese Yen (JPY)

The model is trained on the daily exchange rate of each currency with the US Dollar (USD) as the base currency.
